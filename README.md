## About The Program
This is a program "template" that categorizes messages and looks for appropriate responses.

All data is stored and categorized in `data.json`

## Using

1. Clone the repo
```sh
git clone https://github.com/4rlenrey/RespondTo.git
```
2. Run 
```sh
python3 main.py
```
